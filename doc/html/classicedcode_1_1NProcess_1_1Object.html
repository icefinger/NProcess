Detailed Description

Object to be used by inheritence for the multithread processes.

◆ Lock()

void icedcode::NProcess::Object::Lock ( const std::string &  key_)
inline

Easy to use internal mutex method to be used in Run and Process for multi-theading safety. Slower than direct mutex usage, it uses a dictionary to classify the mutexes by user names. It's the dictionary from the singleton NProcess. This method lock the mutex pointed by key_.

key_string naming the mutex.
